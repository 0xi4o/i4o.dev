# i4o.dev

-   Framework: [Remix](https://remix.run/)
-   Components: [Catalyst UI](https://catalyst.i4o.dev/)
-   Styling: [TailwindCSS](https://tailwindcss.com/)
-   Deployment: [Railway](https://vercel.com/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/0xi4o/i4o.dev.remix
  cd i4o.dev.remix
  pnpm install
  pnpm dev
```
